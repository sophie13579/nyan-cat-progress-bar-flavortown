## Installation

### Option 1: Install from CRX File (Easiest)
1. Download the `.crx` file from the link above
2. Open `chrome://extensions/` in your browser
3. Drag and drop the `.crx` file onto the extensions page (file should be in your Downloads folder)
4. Click "Add extension" to confirm
5. The extension is now active

### Option 2: Install from Source
1. Clone or download the repository
2. Go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the project folder
5. The extension will now be active

Browser Compatibility:
Only Chrome at the moment, the extension is not yet on the google extension store.

Performance:
This extension has minimal effect on the performance!
