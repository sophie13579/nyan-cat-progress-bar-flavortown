## Installation

### Install from Source
1. Clone or download the repository (Code => Download Zip)
2. Unzip the zipped file (the name should be `nyan-cat-progress-bar-flavortown-dev.zip`)
2. Go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the project folder
5. The extension will now be active

Browser Compatibility:
Only Chrome at the moment, the extension is not yet on the google extension store.

Performance:
This extension has minimal effect on the performance!
